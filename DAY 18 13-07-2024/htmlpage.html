HTML Introduction

  • HTML is the standard markup language for creating Web pages

  3. <h3>:This is third largest heading

  5. <h5>:This is fifth largest heading
  6. <h6>:This is smallest heading
Block level & Inline Elements

Block level & Inline Elements

BLock Level Elements:
Block level elements are elements which accupies full space and which always starts on new line
Ex: heading tag, paragraph tag, tabletag

Inline Elements:
Inline elements are the elements which doesn't starts on new line and which takes necessary space
Ex: Anchor tag, mark tag, bold tag, italic tag, strong tag, emphasized tag, superscript,subscript and address tag
